A mini-webgram is a small version of a normal Webgram instance, used as an overview of the whole drawing, enabling the user to pan and zoom to certain areas that are not displayed by the main webgram. A webgram is paired with a mini-webgram using Webgram#setMiniWebgram.
Defined in: mini-webgram.js.

Parameters:
