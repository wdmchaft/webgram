A base class for all the polygonal drawing elements.
Defined in: poly-element.js.

Class Detail
Webgram.DrawingElements.PolyElement(id, points)
Parameters:
{String} id
a given identifier, can be null
{Array} points
a list of Webgram.Geometry.Point objects that form the shape of this polygonal element
Field Detail
{Boolean} closed

true if the polygonal element is a closed shape, false otherwise. For example, a polygon is closed, while a polyline is not.

{Number} maxPointCount

The maximal number of points that the polygonal element is allowed to have.

{Webgram.Event} onAddPoint

An event that is triggered when a poly point is added to this polygonal element.
Handlers receive the following arguments: (index, point)

{Webgram.Event} onRemovePoint

An event that is triggered when a poly point is removed from this polygonal element.
Handlers receive the following arguments: (index, point)

{Number} getLineTolerance()

Returns the tolerance (the thickness) of the edges of this element. This tolerance is used to determine if a point is over the edges or not. Zooming affects the value of this tolerance.

(could be overridden)

Returns:
{Number} the line tolerance
{Webgram.Geometry.Poly} getLineTolerancePoly()

Returns a polygon that wraps all the edges of this element. This polygon is directly used to determine if a point is over the edges of this element.

(should not be overridden)

Returns:
{Webgram.Geometry.Poly} the line tolerance polygon
