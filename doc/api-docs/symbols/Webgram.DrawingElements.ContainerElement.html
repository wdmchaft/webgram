Method Detail
addDrawingElement(drawingElement, where)

Adds a drawing element to the container.

Parameters:
{Webgram.DrawingElement} drawingElement
the drawing element to add
{any} where
indicates where to add the element:
  • if a Webgram.DrawingElement is given, the element is added after that element
  • if a number is given, the element is added at that position
  • if null or undefined is given, the element is added at the end
drawElements()

Draws the children of this container element onto the canvas, by calling their respective drawing methods. See Webgram.DrawingElements.ContainerElement#drawElements for an explanation.

drawSelf()

The drawing procedure of a container element is divided into two parts: the drawing of the container itself (this method) and the drawing of the children, which is done by a call to Webgram.DrawingElements.ContainerElement#drawElements.
