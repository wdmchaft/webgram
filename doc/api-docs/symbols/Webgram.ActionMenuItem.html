Class Webgram.ActionMenuItem

Represents clickable menu item around a drawing element. instances of this class can be used to facilitate actions like copy, paste, duplicate etc.
Defined in: action-menu-item.js.

Field Detail
align

This field indicates the alignment of the menu item relative to the drawing element. It's a string made of two characters:

  • 'l', 'c' or 'r' - left, center or right, respectively, for the horizontal alignment
  • 't', 'c' or 'b' - top, center or bottom, respectively, for the vertical alignment

callback

When the user clicks on the action menu item (read: activates the menu item), this is the function that will be called with the menu item as this. If you need to force an activation, call Webgram.ActionMenuItem#activate.

icon

A menu item is in fact an icon painted around the element. This field represents the name of the image loaded into the Webgram.ImageStore to draw as icon. The default implementation of draw requires this point to a valid image.

{Webgram.Event} onActivate

An event that is triggered when the menu item is activated (either by the user or by a call to Webgram.ActionMenuItem#activate).
Handlers receive no arguments.

Method Detail
activate()

Performs the action associated with this menu item. This is basically called when the user clicks on the item. The associated action consists of a call to callback and a trigger of the onActivate event.
