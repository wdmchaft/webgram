Class Webgram.ImageStore

A store for JavaScript Image objects. This class has a singleton object, Webgram.imageStore that should be used when loading and drawing images with webgram.
Defined in: image-store.js.

Class Detail
Webgram.ImageStore(webgram)
Parameters:
webgram
Method Detail
{Image} get(name)

Returns the image having the given name. If no such image exists (whether because it hasn't been loaded yet, or simply because it never was registered with the store), null is returned.

Parameters:
{String} name
the name of the image
Returns:
{Image} the image with the given name, or null if image was not found
load(name, url, onLoad)

Creates a JavaScript image object inside the store and assigns it a name (an alias). If another image with the given name already exists, it will be overwritten. This method loads the image asynchronously; therefore, the image will not be available right when the call returns.

Parameters:
{String} name
the name to assign to the image
{String} url
the URL of the image to load
{Function} onLoad
a function to be called when the image is loaded, or null/undefined; the function will be called with name, url and image as arguments
