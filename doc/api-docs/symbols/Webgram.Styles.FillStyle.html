Class Webgram.Styles.FillStyle

Represents a style used to fill the shapes on the canvas. Any of the following arguments can be omitted (or passed as null), in which case the default values will be used for that attribute.
Defined in: styles.js.

Class Detail
Webgram.Styles.FillStyle(colors, gradientPoint1, gradientPoint2, gradientRadius1, gradientRadius2)
Parameters:
{Array} colors
a list of colors to use; if a single color is wanted, it can be passed directly (i.e. not enclosed in an array)
{Webgram.Geometry.Point} gradientPoint1
the start gradient point; no gradient will be used if this is omitted
{Webgram.Geometry.Point} gradientPoint2
the end gradient point; if not null a linear gradient will be used, otherwise, if gradientPoint1 is not null, a radial gradient will be used
{Number} gradientRadius1
the start radius of the radial gradient
{Number} gradientRadius2
the end radius of the radial gradient
Field Detail
{Array} colors

A list of colors to use. If a single color is used, this field is set directly to that color instead of being wrapped inside an array.

{Webgram.Geometry.Point} gradientPoint1

The start gradient point. This is set to null if no gradient is used.

{Webgram.Geometry.Point} gradientPoint2

The end gradient point. This is set to a non-null point, if a linear gradient is used.

{Number} gradientRadius1

The start radius of the radial gradient. This is set to null if a linear gradient or no gradient is used.

{Number} gradientRadius2

The end radius of the radial gradient. This is set to null if a linear gradient or no gradient is used.
