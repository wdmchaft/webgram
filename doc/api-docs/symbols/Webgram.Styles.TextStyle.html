Class Webgram.Styles.TextStyle

Represents a style used to draw the text on the canvas. Any of the following arguments can be omitted (or passed as null), in which case the default values will be used for that attribute.
Defined in: styles.js.

Class Detail
Webgram.Styles.TextStyle(color, font, size, bold, italic, justify)
Parameters:
{String} color
the color to be used to draw the text
{String} font
the font name to be used to draw the text
{Number} size
the size of the text
{Boolean} bold
whether to draw a bold text or not
{Boolean} italic
whether to draw the text in italics or not
{Array} justify
an array of two letters; the first one indicates the horizontal alignment and is one of 'l', 'c' or 'r', for left, center or right alignment, respectively; the second one indicates the vertical alignment and is one of 't', 'c' or 'b', for top, center or bottom alignment, respectively
Field Detail

Whether the text is drawn in italics or not.

An array of two letters. The first one indicates the horizontal alignment and is one of 'l', 'c' or 'r', for left, center or right alignment, respectively. The second one indicates the vertical alignment and is one of 't', 'c' or 'b', for top, center or bottom alignment, respectively.

{Number} size
